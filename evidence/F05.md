## 验收标准覆盖

| 验收标准 | 实现方式 |
|----------|----------|
| 「更多」BottomSheet 新增「设置」选项 | bottom_nav.dart 添加 Settings ListTile，icon=Icons.settings |
| SettingsPage：退出登录按钮 + 通知开关 | SwitchListTile（到期任务提醒）+ ListTile（退出登录） |
| 退出登录 -> 清除 token -> 跳转 LoginPage | ref.read(authProvider.notifier).logout() -> context.go('/login') |
| 通知开关打开 -> 请求系统通知权限 | enableNotifications() 调用 requestPermission() |
| 通知开关关闭 -> 取消定时任务 | disableNotifications() 调用 cancelAll() |
| 同一到期任务在一天内最多触发一次本地通知（去重） | SharedPreferences 存储 `{date}:{entryId}` 去重集合 |
| App 启动时（通知已启用）-> 查询 GET /entries?due=today -> 显示本地通知 | main.dart addPostFrameCallback -> initService -> checkDueTasksOnStartup |
| 点击通知 -> 打开 App 到任务页 | NotificationResponse payload='/tasks' + onNotificationTap 回调 |
