## 验收标准覆盖

| 验收标准 | 实现方式 |
|----------|----------|
| GoalsPage 点击目标卡片 → push 到 GoalDetailPage | `_handleGoalTap` 改为 `context.push('/goals/${goal.id}')` |
| 显示目标标题、描述、进度百分比（进度环） | `_buildGoalHeader` 展示标题+描述+ProgressRing |
| 里程碑列表：显示所有里程碑，可勾选完成/取消完成 | `_buildMilestoneItem` + `_handleToggleMilestone` |
| 添加里程碑：底部弹窗输入标题 → POST | FAB → `_showAddMilestoneDialog` → `createMilestone` |
| 删除里程碑：滑动删除确认 → DELETE | Dismissible + confirmDismiss 弹窗 |
| 关联条目列表：展示已关联的 entries | `_buildLinkedEntriesSection` + `fetchLinkedEntries` |
| loading/empty/error 三态处理 | `_buildBody` 三态分支 |

## API 端点使用

- `GET /goals/{id}` — fetchGoalDetail
- `GET /goals/{id}/milestones` — fetchMilestones
- `POST /goals/{id}/milestones` — createMilestone
- `PUT /goals/{id}/milestones/{mid}` — updateMilestone
- `DELETE /goals/{id}/milestones/{mid}` — deleteMilestone
- `GET /goals/{id}/entries` — fetchLinkedEntries (fetchGoalEntries)
