# F05 — Project 卡片 + 进度条

## Task Snapshot

- **ID**: F05
- **Module**: tasks
- **Priority**: high
- **Depends on**: F03 (子 Tab), F04 (DecisionCard 模式)

## Acceptance Criteria

1. project 类型条目在任务列表中渲染为 ProjectCard
2. ProjectCard 显示标题、子任务进度条（X/Y 完成）、最近更新时间
3. 点击 ProjectCard 展开子任务列表（从 store 中 filter parent_id）
4. 「项目」子 Tab 下 ProjectCard 以 2 列网格布局展示
5. 「全部」子 Tab 下 ProjectCard 以紧凑行布局展示
6. 无子任务的 project 显示「暂无子任务」
7. 进度条颜色：完成>80% 绿色，30-80% 蓝色，<30% 灰色
8. 进度数据来自 GET /entries/{id}/progress API（getProjectProgress）
9. 展开的子任务列表来自前端已加载数据，不一致时标注「已加载 N 条」
10. 进度 API 超时或失败时：卡片正常显示标题和时间，进度条显示灰色「加载失败」
