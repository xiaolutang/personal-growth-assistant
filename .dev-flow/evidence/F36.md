- API 失败不阻塞（catch + null）
- 后端 progress_delta 语义:
  - 仅 tag_auto + start_date + end_date 时计算
  - 使用日历周期 cutoff（上周一 / 本月1日）
  - count/checklist 类型 delta 为 None
  - goals 列表仅含活跃目标，completed_count 单独统计
