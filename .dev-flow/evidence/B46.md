**Design decisions:**
- count 进度 = linked_entries_count / target_value
- checklist 进度 = checked items / target_value
- tag_auto 进度 = count_entries_by_tags_in_range(有 start/end) 或 count_entries_by_tags(无日期) / target_value
- 100% 自动完成，进度下降不自动回退
- progress-summary: goals 列表仅含活跃目标，completed_count 单独统计
- progress_delta: 仅 tag_auto + start_date + end_date 时计算，用日历周期（上周一/本月1日）作为 cutoff
- tag_auto 详情页通过 `_list_tag_auto_entries` 返回匹配条目列表
