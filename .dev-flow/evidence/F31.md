### Acceptance Criteria

1. EntryDetail 关联面板区域显示「知识上下文」卡片
2. 缩略图使用 ReactFlow 渲染 nodes 和 edges
3. 中心概念节点（条目 tags）高亮显示
4. 点击节点跳转到 /graph?focus=concept_name，GraphPage 支持 ?focus= 参数
5. 无 tags 时显示「暂无关联概念」引导文案
6. 卡片可折叠，默认展开
7. 移动端图谱缩略图不溢出

### 验收标准逐条验证

1. **知识上下文卡片** ✅ — EntryDetail 页面在 AI 摘要卡片下方、相关条目卡片上方显示「知识上下文」卡片，包含 `Sparkles` 图标
2. **ReactFlow 缩略图** ✅ — `KnowledgeGraphThumbnail` 组件使用 `@xyflow/react` 的 `ReactFlow` 渲染，自定义 `ThumbnailNode` 节点组件
3. **中心概念高亮** ✅ — `center_concepts` 数组中的节点通过 `isCenter=true` 标记，渲染为黄色边框 + ring 高亮
4. **点击跳转** ✅ — 点击缩略图节点调用 `navigate('/graph?focus=concept_name')`，GraphPage 通过 `useSearchParams` 读取 `focus` 参数并自动选中+高亮对应节点
5. **空态引导** ✅ — `nodes.length === 0` 时显示 `Compass` 图标 + "暂无关联概念" + "添加标签可自动生成知识图谱关联" 引导文案
6. **可折叠** ✅ — `knowledgeContextExpanded` 状态控制展开/收起，Header 区域 `onClick` 切换，显示 `ChevronUp`/`ChevronDown` 图标，默认展开
7. **移动端适配** ✅ — 缩略图容器 `h-48 md:h-56` + `overflow-hidden rounded-lg`，不溢出
