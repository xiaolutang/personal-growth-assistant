## 设计约束

- 使用 Python `string.Template` 占位符（`$variable` 格式）
- Chart.js CDN: `https://cdn.jsdelivr.net/npm/chart.js`
- 模板不做转义逻辑，所有数据由 `report_generator.py` 预处理
- HTML 自包含（内联 CSS，无外部样式依赖）
- 离线时 Chart.js 不可加载，canvas 隐藏，fallback 文字替代
- 靛蓝主色调 `#6366F1`
