## Acceptance Criteria

1. 创建 count 类型目标 → 201 + 正确字段
2. 创建 checklist 类型目标 → 201 + checklist_items 有 id/title/checked
3. 创建 tag_auto 类型目标 → 201 + auto_tags 字段
4. tag_auto 缺少 auto_tags → 422
5. checklist 缺少 checklist_items → 422
6. 列出目标 → 返回数组
7. 按 status=active 过滤 → 只返回 active
8. 获取目标详情 → 含 linked_entries_count
9. 更新目标状态 active→completed → status 变为 completed
10. 删除 active 目标 → 拒绝（仅 abandoned 可删）
11. 关联条目到 count 目标 → current_value +1
12. 取消关联条目 → current_value -1
13. 切换 checklist 项 → checked 翻转
14. 进度汇总 → 返回 active_count + completed_count
15. 未认证请求 → 401
