### Changes

1. **api.ts** — 类型重命名（FeedbackIssue→FeedbackItem, FeedbackResponse→FeedbackSubmitResponse），新增 FeedbackListResponse 类型；新增 getFeedbackList() 和 getFeedbackDetail() 函数
2. **FeedbackButton.tsx** — 改造为双 Tab 面板：
   - Tab 头：提交反馈 / 我的反馈 / 关闭
   - 提交 Tab：保持原有表单，成功后自动切到列表 Tab
   - 列表 Tab：加载状态、错误态（重试按钮）、空态引导、反馈列表（title+status+time）
   - status pending 显示黄色「待处理」，reported 显示绿色「已上报」
   - 成功后只切 Tab 不主动加载，由 useEffect 统一加载避免双请求
3. **FeedbackButton.test.tsx** — 新增 3 个测试（列表展示+状态颜色、空列表引导、加载失败+重试），更新成功路径测试（断言 getFeedbackListMock 调用 1 次）
4. **api.feedback.test.ts** — 新增 getFeedbackList（成功+503）和 getFeedbackDetail（成功+404）测试
5. **api.generated.ts** — 重新生成，包含 GET /feedback 和 GET /feedback/{feedback_id} 端点
