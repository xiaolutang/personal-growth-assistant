# F07b — Flutter formatDate 统一

## 变更摘要

创建统一的 `DateFormatter` 工具类，替换 6 处页面内重复且不一致的 `_formatDate` / `_formatTime` 实现。

## 变更文件

| 文件 | 操作 |
|------|------|
| `mobile/lib/utils/date_formatter.dart` | 新建 — 统一日期格式化工具类 |
| `mobile/lib/pages/goals_page.dart` | 修改 — `_formatDate` → `DateFormatter.formatShortDate` |
| `mobile/lib/pages/goal_detail_page.dart` | 修改 — `_formatDate` → `DateFormatter.formatShortDate` |
| `mobile/lib/pages/notes_page.dart` | 修改 — `_formatDate` → `DateFormatter.formatShortDate` |
| `mobile/lib/pages/inbox_page.dart` | 修改 — `_formatTime` → `DateFormatter.formatRelative` |
| `mobile/lib/pages/entry_detail_page.dart` | 修改 — `_formatDate` → `DateFormatter.formatFullDate`，移除 `intl` 依赖 |
| `mobile/lib/pages/review_page.dart` | 无变更 — 无独立日期格式化方法 |
| `mobile/test/utils/date_formatter_test.dart` | 新建 — 25 个测试用例 |

## DateFormatter API

| 方法 | 用途 | 格式示例 |
|------|------|----------|
| `formatRelative()` | 相对时间 | 刚刚 / 5分钟前 / 3小时前 / 2天前 / 5月1日 14:30 |
| `formatShortDate()` | 月/日 | 5月1日 / 2025年12月25日(跨年) |
| `formatFullDate()` | 完整日期时间 | 2026-05-01 14:30 |

## 格式变更说明

- **notes_page**: M/D → X月X日（统一为中文格式）
- **inbox_page**: 超过7天从 `M/D HH:mm` 改为 `X月X日 HH:mm`（中文统一）
- 其余页面格式不变
