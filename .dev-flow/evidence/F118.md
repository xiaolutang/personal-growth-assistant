### Acceptance Criteria

1. onboarding_completed=false 时，不显示静态弹窗 OnboardingFlow
2. onboarding_completed=false 时，Home 页 PageChatPanel 自动展开（非折叠）
3. PageChatPanel 显示日知欢迎消息作为初始 assistant 消息（无需用户触发）
4. 欢迎消息内容：日知自我介绍 + 示例短语
5. suggestions 显示 onboarding 专属建议：记灵感、做任务、记笔记
6. pageContext 包含 is_new_user: true
7. onboarding_completed=false 时，FloatingChat 在首页隐藏（避免双入口混淆）
8. 用户首次对话完成状态机：用户发送消息 → SSE 流式响应结束 → updateMe → key 重挂载
9. updateMe 失败不阻塞，下次仍为 onboarding 模式
10. updateMe 防重复：boolean flag 保证只调用一次
11. onboarding_completed=true 后，PageChatPanel 恢复正常模式
12. 窗口宽度 < 640px 时不溢出
13. 用户可关闭/收起 chat panel

### Files Modified

- `frontend/src/components/PageChatPanel.tsx` — 新增 greetingMessage + onFirstResponse props
- `frontend/src/pages/Home.tsx` — onboarding 状态管理 + PageChatPanel 双模式

- `frontend/src/services/api.ts` — AIChatContext 新增 is_new_user 字段
