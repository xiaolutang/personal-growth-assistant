# B03 — Backend HybridSearchService 注入

## 任务摘要

修复 EntryService 中 HybridSearchService 每次新建实例的问题，通过构造函数注入 HybridSearchService。同时替换 hybrid_search.py 中 `asyncio.get_event_loop()` 为 `asyncio.get_running_loop()`。

## 改动文件

| 文件 | 改动 |
|------|------|
| `backend/app/services/hybrid_search.py` | `asyncio.get_event_loop()` -> `asyncio.get_running_loop()` |
| `backend/app/services/entry_service.py` | 构造函数新增可选 `hybrid_search` 参数 + `set_hybrid_search` setter + `search_entries`/`get_related_entries` 使用注入实例 |
| `backend/app/routers/deps.py` | `get_entry_service()` 创建时注入 `HybridSearchService` + 延迟注入逻辑 |
| `backend/tests/unit/services/test_hybrid_search.py` | 新增 `TestB03AsyncioApiUsage` 验证源码不含 `get_event_loop` |
| `backend/tests/unit/services/test_entry_service.py` | 新增 `TestB03HybridSearchInjection` 5 个测试 |
| `backend/tests/unit/services/test_related_entries.py` | 适配注入模式，移除 `patch("...HybridSearchService")` |
