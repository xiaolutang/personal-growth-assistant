# Allowed Origin Preflight (OPTIONS)

# access-control-allow-headers: Accept, Accept-Language, Authorization, Content-Language, Content-Type, X-Request-ID, X-UID

# Disallowed Origin
curl -sI -H "Origin: http://evil.example.com" -X OPTIONS http://localhost:18765/health

# TRACE Method Preflight
