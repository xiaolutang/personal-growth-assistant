# F04 - Notes 搜索防抖

## 任务
在 notes_page.dart 的搜索中接入 Debouncer，搜索输入 300ms 防抖。

## 修改文件
- `mobile/lib/pages/notes_page.dart`

## 修改内容

### 1. 添加 import
```dart
import '../utils/debouncer.dart';
```

### 2. 添加 Debouncer 实例
```dart
final _debouncer = Debouncer();
```

### 3. 修改搜索逻辑 (_handleSearch)
- 有输入时：`_debouncer.run(() => searchNotes(query))` — 300ms 防抖
- 空查询时：`_debouncer.immediateRun(() => fetchNotes())` — 立即执行

### 4. 清空按钮逻辑
```dart
_searchController.clear();
_debouncer.immediateRun(() => ref.read(notesProvider.notifier).fetchNotes());
```

### 5. dispose 清理
```dart
@override
void dispose() {
  _debouncer.dispose();
  _searchController.dispose();
  super.dispose();
}
```
