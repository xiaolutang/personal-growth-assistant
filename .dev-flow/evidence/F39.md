## Implementation Summary

### 新增文件

### 修改文件
- `frontend/src/stores/chatStore.ts` — 新增 `pageExtra: Record<string, string | number> | null` 状态 + `setPageExtra` 方法
- `frontend/src/components/FloatingChat.tsx` — 集成 PageSuggestions，合并 pageExtra 到 pageContext.extra，路由变化清空 pageExtra
- `frontend/src/pages/Explore.tsx` — useEffect 同步 activeTab/searchQuery 到 chatStore.pageExtra

### 功能要点
1. **PageSuggestions 组件**: 按 page_type 渲染对应建议 chips（home/explore/entry/review/graph），点击自动填入输入框
2. **隐藏逻辑**: 对话有消息时（hasMessages=true）不显示 chips
3. **chatStore.pageExtra**: 页面组件主动写入本地状态，FloatingChat 在发送 API 时合并到 pageContext.extra
4. **Explore 页同步**: useEffect 监听 activeTab/searchQuery 变化，调用 setPageExtra；离开页面时自动清空
5. **路由变化清理**: FloatingChat 路由变化时自动清空 pageExtra
