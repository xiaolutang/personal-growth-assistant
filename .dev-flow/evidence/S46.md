### 实现文件

### 核心组件

1. **AgentMetrics** — 指标采集器
   - `record_tool_call(tool_name, correct, latency_ms, tokens_used, params_correct)` — 记录单次工具调用
   - `record_ask_user(was_necessary)` — 记录追问行为
   - `get_tool_selection_accuracy()` — 工具选择准确率
   - `get_param_extraction_accuracy()` — 参数提取准确率
   - `get_average_latency()` — 平均延迟（ms）
   - `get_total_tokens()` — token 消耗总量
   - `get_ask_user_accuracy()` — 追问准确率
   - `get_tool_breakdown()` — 按工具名分组统计
   - `get_summary()` — 所有指标汇总
   - `set_weekly_tokens()` / `get_previous_week_tokens()` — 历史 token 对比

2. **AlertRule** — 告警规则定义（dataclass）
   - `name`, `condition`, `message`, `severity`

3. **AlertEngine** — 告警引擎
   - 内置 4 条规则：工具选择准确率<85%、参数提取准确率<85%、延迟>3000ms、token突增>2x
   - `add_rule()` — 自定义规则注册
   - `check_rules(metrics)` — 检查所有规则，返回 Alert 列表

4. **MetricsReport** — 报告生成器
   - `daily_report(metrics, date)` — 日报
   - `weekly_report(metrics, week_start)` — 周报
   - 包含：指标汇总、告警列表、建议动作
   - 输出为 JSON 可序列化的 dict
