## 关键设计决策

1. **OpenAICompatibleChatModel**: 项目未依赖 langchain-openai，创建 BaseChatModel 子类，利用现有 AsyncOpenAI client 并实现 bind_tools，将 LangChain Tool schema 转换为 OpenAI function calling 格式。

2. **手动 Tool 执行**: Tool 函数使用 keyword-only 参数 (dependencies, user_id)，不兼容 ToolNode 标准注入。改为在 _tool_node 中手动遍历 tool_calls 并调用 tool.coroutine() 注入依赖。

3. **ReActAgentState**: 扩展 MessagesState 添加 iteration_count，在 _tool_node 递增，条件边检查上限。

4. **双条件边路由**: _route_after_agent (agent→tools/END) + _route_after_tools (tools→agent/END)
