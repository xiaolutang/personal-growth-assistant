## 实现要点

1. **离线检测**: 在 handleBatchDelete/handleBatchCategory 中通过 `navigator.onLine` 检测网络状态
2. **离线提示**: 离线全量成功时 toast 显示"离线：已标记删除 N 条，联网后自动同步"；UI 显示浮动离线提示条
3. **同步完成刷新**: 通过 `subscribeSyncProgress` 订阅同步完成事件，触发 `onSyncCompleted` 回调刷新 Explore 列表
4. **部分失败条目**: `failedItems` 数组记录失败条目的 id 和 title，通过浮动面板展示具体失败条目
5. **清除失败提示**: `clearFailedItems` 方法供用户手动关闭失败面板
