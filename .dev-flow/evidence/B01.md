### Design Decision: redirect 信号机制

redirect 行为通过 redirect_to_chat 工具实现（非确定性兜底）：
- command prompt 引导模型在闲聊时调用 redirect_to_chat
- 代码层 intercept 该 tool_call → redirect SSE
- 不加确定性兜底（"无 tool_call 的文本 → redirect"），因为 command prompt 也允许直接回答和简短澄清
- 如果模型不遵循 prompt（闲聊时不调 redirect_to_chat），结果是 suboptimal（闲聊显示为 content），但不是 broken（合法回答不会被错误 redirect）
