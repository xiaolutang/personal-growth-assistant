# F141: 搜索结果内容摘要

## 验收条件

| # | 条件 | 状态 |
|---|------|------|
| 1 | TaskCard 搜索模式下显示 content_snippet 摘要（截取前 100 字符） | PASS |
| 2 | 搜索关键词在摘要中高亮 | PASS |
| 3 | UTF-8 字符边界安全（不截断多字节字符） | PASS |
| 4 | 非搜索模式下 TaskCard 行为不变（不展示摘要） | PASS |
| 5 | build 无 TS 错误 | PASS |
| 6 | vitest 覆盖摘要展示和高亮 | PASS |

## 实现详情

### 修改文件
- `frontend/src/components/TaskCard.tsx`
  - 新增 `safeTruncate(text, maxLen)` 工具函数：使用 `Array.from()` 按码点拆分，避免在 surrogate pair 中间截断
  - 搜索模式下计算 `snippetText`：优先使用 `content_snippet`，回退到 `content`
  - snippet 使用 `HighlightText` 组件渲染关键词高亮
  - 非搜索模式（无 `highlightKeyword`）不展示 snippet

### 新增测试文件
- `frontend/src/components/__tests__/TaskCard.snippet.test.tsx`（18 个用例）
  - `safeTruncate` 单元测试：短文本不截断、长文本截断、CJK 安全、emoji 安全、混合字符
  - 摘要展示：搜索模式显示、非搜索模式不显示、空字符串不显示、优先 content_snippet、回退 content、都为空不渲染
  - 关键词高亮：单匹配高亮、多匹配高亮、大小写不敏感、无匹配纯文本
  - UTF-8 安全：CJK 不截断、emoji 不截断
