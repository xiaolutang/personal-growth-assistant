## 日期比较逻辑

采用 UTC 日期字符串比较，与后端 B105 一致：
```typescript
const plannedDateStr = task.planned_date.split("T")[0]; // "2024-06-15"
const todayStr = new Date().toISOString().split("T")[0]; // "2024-06-15"
const isOverdue = plannedDateStr < todayStr;
const isDueToday = plannedDateStr === todayStr;
```
