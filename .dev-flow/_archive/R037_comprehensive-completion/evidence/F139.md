## 实现细节

### 真实无任务（allTasks.length === 0）
- 使用 `ClipboardList` 图标（h-12 w-12 opacity-20）作为空状态插画
- 引导文案："还没有任务，开始记录你的第一个任务吧"
- 快速创建入口："去创建任务" 按钮，调用 `navigate("/")` 跳转首页
- 全部通过 TaskList 组件的 `emptyIcon`/`emptyMessage`/`emptyAction` props 传递

### 筛选无结果（allTasks.length > 0 && filteredTasks.length === 0）
- 使用 `SearchX` 图标（h-10 w-10 opacity-30）
- 提示文案："当前筛选条件下没有匹配的任务"
- "清除筛选" 按钮，调用 `clearFilters()` 重置所有筛选条件
- 同样通过 TaskList 组件的 `emptyIcon`/`emptyMessage`/`emptyAction` props 传递，避免 UI 模式分叉

### 状态判定逻辑
- `isTotallyEmpty = !isLoading && allTasks.length === 0`：真正无任务
- `isFilterEmpty = !isLoading && allTasks.length > 0 && filteredTasks.length === 0`：筛选无结果
- 加载中优先显示 spinner，不显示空状态
