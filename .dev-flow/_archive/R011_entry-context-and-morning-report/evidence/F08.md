- recharts LineChart + ResponsiveContainer(width="100%") 实现趋势折线图
- 趋势卡片使用独立 useState: trendLoading / trendError / trendData，与主报告区域完全隔离
- getReviewTrend 使用 raw fetch + buildAuthHeaders + handleApiResponse（与项目 API 调用模式一致）
- 日/周切换：Badge variant 切换，硬编码 days=7 和 weeks=8
- 空数据：引导文案 + TrendingUp 图标
- 错误：卡片内红色提示文案（不隐藏卡片）
- 平均完成率摘要：trendData.reduce 计算，显示在图表下方 border-t 区域
