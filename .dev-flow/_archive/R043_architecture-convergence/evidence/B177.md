## _conn() 上下文管理器说明

```python
@contextmanager
def _conn(self):
    conn = self._get_conn()
    try:
        yield conn
        conn.commit()
    except Exception:
        conn.rollback()
        raise
    finally:
        conn.close()
```

- 正常退出：自动 commit + close
- 异常退出：自动 rollback + close
- `_init_db()` 保留手动管理，因为需要统一 commit 所有 DDL
