## 技术细节

### sync_feedbacks 并发化
- 使用 `asyncio.Semaphore(10)` 限制并发上限
- `asyncio.gather(*[_sync_one(fb, http_client)], return_exceptions=True)` 并发请求
- `_sync_one` 返回 `(was_synced, was_updated)` 元组区分计数
- 单条超时/HTTP 错误/未知状态均不影响其他反馈

### AnalyticsService 连接管理
- `ensure_table` 和 `record_event` 均改用 `with self._sqlite._conn() as conn:`
- 消除了手动 `get_connection()` + `conn.commit()` + `conn.close()` 模式
- 移除了 `sqlite3` import 和 `_get_conn` 方法
