### implementation_details

1. **CompletionPrompt 组件**: 固定定位在底部，显示「写个复盘？」提示和两个按钮。写复盘时调用 `createEntry({ type: "reflection", title: ..., parent_id: ... })`，创建成功后 toast 提示并跳转。
2. **TaskCard 集成**: 添加 `dismissedPromptIds` state，当 task.status === "complete" 且未跳过时渲染 CompletionPrompt。跳过后将 task.id 加入 Set，防止重复弹出。
3. **Explore 页面 entry_id 处理**: 读取 URL 的 `entry_id` 参数，在数据加载完成后找到目标条目并自动导航到详情页 `/entries/{entry_id}`。处理后清除 URL 参数。
4. **错误处理**: 创建 reflection 失败时 toast error 提示，不关闭提示框（允许重试）。创建中显示 loading 状态，按钮禁用。
