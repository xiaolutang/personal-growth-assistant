# F06 - 探索 Tab 精简

## 变更摘要

从探索页移除 project/decision tab，保留搜索全类型。搜索结果中 task/decision/project 跳转到任务页。

## 修改文件

| 文件 | 变更 |
|------|------|
| `frontend/src/pages/explore/utils.ts` | TABS 从 7 个精简为 5 个（移除 project/decision），EXPLORE_CATEGORIES 从 6 个精简为 4 个（只含 inbox/note/reflection/question），移除 Folder/Scale 图标 import |
| `frontend/src/pages/explore/useExploreSearch.ts` | 搜索模式 filteredTasks 不再过滤类型，全类型混合展示；非搜索模式仍按 EXPLORE_CATEGORIES 过滤 |
| `frontend/src/pages/Explore.tsx` | 添加搜索模式 onCardClick 回调，task/decision/project 跳转 /tasks?tab={category}，inbox/note/reflection/question 跳转 /entries/{id}；导入 EXPLORE_CATEGORIES 和 Task 类型 |
| `frontend/src/App.tsx` | /projects 重定向从 `/explore?type=project` 改为 `/tasks?tab=project` |
| `frontend/src/pages/Home.tsx` | "记决策"快捷操作从 `/explore?type=decision` 改为 `/tasks?tab=decision` |
| `frontend/src/components/TaskList.tsx` | 添加 `onCardClick` prop，透传给 TaskCard/DecisionCard/ProjectCard |
| `frontend/src/components/TaskCard.tsx` | 添加 `onClickOverride` prop，覆盖默认导航行为 |
| `frontend/src/pages/tasks/DecisionCard.tsx` | 添加 `onClickOverride` prop，覆盖默认点击行为 |
| `frontend/src/pages/tasks/ProjectCard.tsx` | 添加 `onClickOverride` prop，覆盖默认点击行为 |
