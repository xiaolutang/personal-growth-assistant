## Implementation

### 1. vite-plugin-pwa 配置
- **File**: `frontend/vite.config.ts`
- 安装 `vite-plugin-pwa` dev dependency
- 配置 VitePWA 插件：registerType=autoUpdate, manifest, workbox
- manifest: name/short_name/description/theme_color/background_color/display/icons
- workbox: navigateFallback=index.html, API runtimeCaching=NetworkFirst
- scope 自动为 base path `/growth/`

### 2. PWA 图标
- **Files**: `frontend/public/pwa-192x192.png`, `frontend/public/pwa-512x512.png`
- 使用 Python PIL 生成占位图标（靛蓝色背景 + 白色圆角矩形）

### 3. 安装提示 Hook
- **File**: `frontend/src/lib/usePWAInstall.ts` (NEW)
- 监听 `beforeinstallprompt` 事件
- 提供 `canInstall` 状态和 `promptInstall()` 方法
- 安装后自动隐藏提示

### 4. Header 安装按钮
- **File**: `frontend/src/components/layout/Header.tsx`
- canInstall=true 时显示 Download 图标按钮
- 点击触发 promptInstall()
