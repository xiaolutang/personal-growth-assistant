# B43: 条目知识上下文 API

## Task Snapshot

- **ID**: B43
- **Name**: 条目知识上下文 API
- **Phase**: P1
- **Module**: knowledge-graph
- **Priority**: high
- **Verification**: L2

### Acceptance Criteria

1. GET /entries/{id}/knowledge-context 返回 { nodes, edges, center_concepts }
2. nodes 包含 id, name, category, mastery, entry_count
3. edges 包含 source, target, relationship
4. center_concepts 为该条目的 tags 列表
5. 无 tags 时返回空子图
6. Neo4j 不可达时降级为 SQLite 标签共现图
7. 按 user_id 隔离
8. 单元测试覆盖正常/降级/空数据场景

### Test Tasks

- 正常：有 tags 的条目返回非空 nodes 和 edges
- 正常：center_concepts 与条目 tags 一致
- 边界：无 tags 条目返回空子图
- 边界：Neo4j 不可达时降级为 SQLite 数据
- 异常：不存在的条目返回 404
- 异常：未认证返回 401
