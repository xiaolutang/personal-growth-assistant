- 状态筛选 tabs: active/completed/abandoned

- 详情页:

  - tag_auto: 追踪标签 badges + 匹配条目列表（自动从 API 获取）
  - 归档/重新激活按钮
- 响应式网格: sm:grid-cols-2 lg:grid-cols-3
- 空状态引导: "设定一个目标开始追踪"
