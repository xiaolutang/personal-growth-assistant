## Provider 详情

### 1. NotesProvider
- **NotesState**: entries, isLoading, error, searchQuery
- **NotesNotifier**: fetchNotes(), searchNotes(query), clearSearch()
- 使用 `apiClient.fetchEntries(type: 'note')` 和 `apiClient.searchEntries`
- 搜索时过滤只保留 note 类型

### 2. InboxProvider
- **InboxState**: entries, isLoading, error, newEntryText
- **InboxNotifier**: fetchInbox(), createInboxItem(title), convertCategory(id, newCategory), setNewEntryText(text)
- createInboxItem 使用 `apiClient.createEntry(data: {...})`
- convertCategory 使用 `apiClient.updateEntryCategory`

### 3. ReviewProvider
- **ReviewState**: summary, trends, insights, isLoading, error, selectedPeriod
- **ReviewNotifier**: loadSummary(period), loadTrends(period), loadInsights(), setPeriod(period)
- 使用 `apiClient.fetchReviewSummary`, `apiClient.fetchTrends`, `apiClient.fetchInsights`
- copyWith 支持清除 nullable Map 字段（summary/trends/insights）

### 4. GoalsProvider
- **GoalsState**: goals (List<Goal>), selectedGoal (Goal?), milestones (List<Milestone>), isLoading, error
- **GoalsNotifier**: fetchGoals(status), fetchGoalDetail(id), fetchMilestones(goalId), createMilestone(goalId, data), updateMilestone(goalId, milestoneId, data), deleteMilestone(goalId, milestoneId)
- 包含 Goal 和 Milestone 数据模型（fromJson）
- 使用 `apiClient` 对应的 Goals API 方法
