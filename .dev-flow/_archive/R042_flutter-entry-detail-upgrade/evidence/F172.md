## 新增 API 方法

| 方法 | HTTP | 路径 | 说明 |
|------|------|------|------|
| `updateEntry` | PUT | `/entries/{id}` | 更新条目，返回 SuccessResponse |
| `fetchBacklinks` | GET | `/entries/{id}/backlinks` | 获取反向链接 |
| `fetchEntryLinks` | GET | `/entries/{id}/links?direction=` | 获取关联链接，direction=in/out/both |
| `createEntryLink` | POST | `/entries/{id}/links` | 创建关联，body: target_id + relation_type，返回 201 |
| `deleteEntryLink` | DELETE | `/entries/{id}/links/{linkId}` | 删除关联，返回 204 |
| `fetchKnowledgeContext` | GET | `/entries/{id}/knowledge-context` | 知识上下文（nodes/edges/center_concepts）|
| `generateAISummary` | POST | `/entries/{id}/ai-summary` | AI 摘要（summary/cached/generated_at）|

## 测试覆盖

### 正常响应测试 (7)
- updateEntry 返回 SuccessResponse
- fetchBacklinks 返回 backlinks 列表
- fetchEntryLinks 默认 direction=both
- fetchEntryLinks direction=in 参数传递
- fetchEntryLinks direction=out 参数传递
- createEntryLink 返回 201 + EntryLinkResponse
- deleteEntryLink 返回 204
- fetchKnowledgeContext 返回 nodes/edges/center_concepts
- generateAISummary 返回 summary/cached/generated_at

### 边界场景测试 (6)
- fetchBacklinks 空列表
- fetchEntryLinks direction=invalid 返回 422
- createEntryLink 自关联返回 400
- createEntryLink 重复关联返回 409
- generateAISummary 缓存命中 (cached=true)
- generateAISummary 空内容 summary=null

### 错误处理测试 (10)
- updateEntry 404
- updateEntry 500
- fetchBacklinks 404
- fetchEntryLinks 404
- createEntryLink 404 (目标不存在)
- deleteEntryLink 404
- fetchKnowledgeContext 404
- fetchKnowledgeContext 500
- generateAISummary 404
- generateAISummary 500
