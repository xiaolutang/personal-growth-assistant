# 个人成长助手 - 产品设计文档

> **口号**：过程属于自己
>
> **品牌故事**：在 AI 时代，你的工作成果属于公司，但你的成长过程、学习笔记、决策思考——这些才是真正属于你的财富。

---

## 一、产品定位

### 1.1 目标用户

| 用户类型 | 痛点 | 付费意愿 |
|---------|------|---------|
| 知识工作者 | 知识碎片化、项目多 | 高 |
| 终身学习者 | 学了记不住 | 中 |
| 自由职业者 | 时间管理、项目追踪 | 高 |
| AI 工具使用者 | 数据分散、无法统一管理 | 高 |

### 1.2 核心价值

**记录成长，掌控数据**

- 用户完全掌控自己的数据（本地 Markdown 文件）
- AI 工具可以直接操作数据（MCP 协议）
- 自动整理，智能检索

### 1.3 差异化卖点

| 卖点 | 说明 | 竞品对比 |
|------|------|---------|
| 本地 Markdown 存储 | 数据透明，用户可直接编辑 | Mem/Reflect 是黑盒 |
| 原生 MCP 支持 | Claude Code/Cursor 可直接操作 | 独一份 |
| AI First | 自动整理、智能检索 | 国内产品 AI 弱 |
| 个人成长场景 | 任务+笔记+项目+决策闭环 | 通用笔记无此场景 |

## 三、商业模式

### 3.1 定价策略

| 版本 | 价格 | 功能 |
|------|------|------|
| **免费版** | ¥0 | 基础输入、关键词搜索、本地存储 |
| **专业版** | ¥29/月 | 无限 AI 解析、语义搜索、时间统计 |
| **团队版** | ¥49/人/月 | 专业版 + 云同步 + 协作 |

### 3.2 付费转化路径

```
用户注册
    ↓
免费使用基础功能
    ↓
体验 AI 解析（10次/月免费）
    ↓
遇到限制 → 付费升级
    ↓
深度使用 → 推荐给朋友
```

## 四、用户场景

### 场景1：快速记录想法

> 用户：今天学了 RAG，重点是向量检索和 Embedding
>
> 系统：自动创建笔记，提取标签 #RAG #向量检索 #Embedding，关联到 AI 转型项目

### 场景2：查找历史内容

> 用户：上周关于 RAG 的笔记是什么？
>
> 系统：通过语义搜索找到相关笔记，展示内容

### 场景3：时间统计分析

> 用户：我最近时间花哪了？
>
> 系统：统计最近 30 天的任务完成情况，按项目/类别分组展示

### 场景4：AI 工具集成

> 用户（在 Claude Code 中）：帮我更新 AI 转型项目的进度
>
> Claude Code：通过 MCP 调用个人成长助手，修改对应 Markdown 文件
