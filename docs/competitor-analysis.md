## 四、市场空白分析

### 4.1 竞品无法满足的需求

| 需求 | 竞品现状 | 我们的机会 |
|------|---------|-----------|
| **中文 + AI 原生** | Mem/Reflect 无中文 | 中文原生 |
| **数据透明可控** | 都是黑盒存储 | 本地 Markdown |
| **AI 工具集成** | 不支持 MCP | 原生 MCP 支持 |
| **个人成长场景** | 通用笔记 | 任务+笔记+项目+决策闭环 |
| **中间价位** | 要么贵要么简陋 | ¥29/月 |

### 4.2 用户真实痛点

来自知乎、即刻、小红书的用户反馈：

- "不知道时间花哪了"
- "笔记记了一堆，找不到"
- "wolai 免费功能开始收费了"
- "Get 笔记 699 买完就吃灰"
- "飞书太重，个人用不起来"
- "想要一个能和 Claude Code 联动的笔记工具"

## 五、我们的差异化策略

### 5.1 核心差异化

```
1. 本地 Markdown 存储
   - 用户完全掌控数据
   - 可以直接编辑文件
   - 支持 Git 版本控制

2. 原生 MCP 支持
   - Claude Code/Cursor 可直接操作
   - 独一份的能力
   - 技术门槛高，竞品难以复制

3. 个人成长场景
   - 不是通用笔记
   - 聚焦任务+笔记+项目+决策
   - 形成成长闭环
```

### 5.2 定位差异

| 产品 | 定位 |
|------|------|
| Notion | All-in-one 工作台（复杂） |
| flomo | 碎片化记录（简单） |
| **我们** | AI First 个人成长管理（聚焦） |
